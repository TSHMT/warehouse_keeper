  <li><h2>遊び方</h2></li>
  <br>
    <p>
      <a>
      プレイヤーはキャラクターを操作して<br>
      黄色の荷物を白印の目的地まで運んであげてください。<br>
      全ての荷物を運ぶとステージクリアになります。<br>
      </a><br>
    </p>
  <li><h2>操作方法</h2></li>
  <br>

      <a>ステージの最初にもどる：スペース</a><br>
      <a>終了：q</a><br>
